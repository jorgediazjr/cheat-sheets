### ELM

#### Install Elm

#### Math Operations
```elm
addTwoIntegers : Int -> Int -> Int
addTwoIntegers number1 number2 = number1 + number2

-- add 25 and 50
addTwoIntegers 25 50
-- result would be 75

-- another way of adding two numbers
addTwoIntegers : Int -> Int -> Int
addTwoIntegers number1 number2 = (+) number1 number2
```
